# AI TEAM PRODUCTIVITY PLATFORM
- AI Team Productivity Platform is a full-stack SaaS application designed to help organizations manage employees, projects, tasks, workloads, and collaboration at scale.
- The system supports multi-role authentication, real-time communication, AI-powered assistance, and subscription-based access, making it suitable for  organizations or single user to manage their projects load.
- This project demonstrates end-to-end product thinking, not just feature implementation.

##  Real-World Problem It Solves
-  Lack of visibility into employee workload.
- Poor task distribution across teams.
- Inefficient project coordination.
- No centralized collaboration system.
- Manual payment & subscription handling.


###  User Roles & Responsibilities
##   Organization Admin
- Create and manage organizations, branches, and employees.
- Monitor employee workload & activity.
- Full system-level access.

##  Project Lead/ Tech Lead
- Create and manage projects.
- Assign members to projects.
- Distribute tasks.
- Track project progress and team performance.
- make edit and view projects.

## Project Memeber/  employee
- assign tasks to member.
- help other member to their task  give ideas through chats and file.
- task view , add,  delete task if that task assignee to you only or tech lead has access to delete.
- accept/ reject  invitation of project.
- Collaborate with team members.
- Use AI assistance for productivity
